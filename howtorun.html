How to run

Option 1: web server

You can put it in your web server, like any static web page.
It does not works directly from disc because the browser forbids to read .html files.

Option 2: node simple web server

  1. Install node js: http://nodejs.org/
  2. Install http-server (run from command line):
    Windows:
    	C:\> npm install -g http-server
    Mac/Linux:
    	$ sudo npm install -g http-server
  3. Run web server from your folder (run from command line):
    	$ cd your/folder/DesignerHowToAngular
	$ http-server -c-1 .
Now you will have a very useful server that avoids any browser cache problems.
